## STEP 1: 씨앗

- **주제**: RAG 기반 사내 AI 비서 만들기 — LLM + 벡터DB + FastAPI로 사내 문서 검색 AI 구축
- **집필 동기**: 기존 RAG 자료가 너무 이론적이다 — 실전 프로젝트로 처음부터 끝까지 만들어보는 자료가 부족
- **저자 경험**: AI가 범용적으로 작성 — "처음 접하는 사람의 전형적 경험"으로 작성
- **핵심 메시지**: "직접 만들어봐야 이해한다" — 환각 체험 → 코드 작성 → 튜닝 실험까지 직접 해보는 여정
- **책의 의도**: RAG 전체 파이프라인 이해 — 문서 파싱 → 청킹 → 임베딩 → 검색 → 답변 생성의 흐름을 개념적으로 이해
- **의도 밖 범위**: Docker/PostgreSQL 세팅 상세, Langfuse/모니터링 상세, CSS/Jinja2 프론트엔드 UI 구현
