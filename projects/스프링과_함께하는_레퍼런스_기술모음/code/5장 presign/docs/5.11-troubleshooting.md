# 5.11 트러블슈팅 체크리스트(403/트리거/완료/권한)

## Presigned 403
- SignatureMismatch일 경우 Content-Type 불일치 여부를 확인합니다.
- 만료 시간 초과는 presigned 발급 후 15분 이내 업로드 여부를 확인합니다.
- Key 규칙 오타는 `original/uuid.ext` 형식을 확인합니다.

## 업로드 성공인데 Lambda 미동작
- S3 이벤트 트리거를 확인합니다 (Prefix: `original/`).
- Lambda 권한에서 S3 호출 가능 여부를 확인합니다.

## Lambda 처리 실패
- 메모리/타임아웃을 상향합니다.
- `sharp` 번들 포함 여부를 확인합니다.
- CloudWatch 로그에서 에러를 확인합니다.

## 완료 저장 실패
- 업로드 후 5초 이상 대기했는지 확인합니다.
- `resized/` 경로에 객체가 생성되었는지 확인합니다.
- 필요 시 Lambda 로그에서 처리 완료 여부를 확인합니다.

## URL 접근 실패
- S3 버킷 퍼블릭 차단 여부를 확인합니다.
- 필요 시 Presigned GET 발급으로 전환합니다.
