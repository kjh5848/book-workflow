# 5.9 Postman 실습 3: 목록 → 상세 → URL 열람으로 최종 확인

> 목표: 목록/상세 API를 호출해 저장된 메타를 확인합니다.

## 1) 목록 조회
- Method는 `GET`입니다.
- URL은 `http://localhost:8080/api/images/list`입니다.

(확인) 경로: postman/5.9-list-response.json
```json
[
  {
    "id": 1,
    "uuid": "uuid",
    "originalUrl": "https://bucket.s3.region.amazonaws.com/original/uuid.jpg",
    "resizedUrl": "https://bucket.s3.region.amazonaws.com/resized/uuid.jpg",
    "fileName": "sample.jpg",
    "createdAt": "2025-01-01 12:00"
  }
]
```
목록 응답 예시입니다.

## 2) 상세 조회
- Method는 `GET`입니다.
- URL은 `http://localhost:8080/api/images/{id}`입니다.

(확인) 경로: postman/5.9-detail-response.json
```json
{
  "id": 1,
  "uuid": "uuid",
  "originalUrl": "https://bucket.s3.region.amazonaws.com/original/uuid.jpg",
  "resizedUrl": "https://bucket.s3.region.amazonaws.com/resized/uuid.jpg",
  "fileName": "sample.jpg",
  "createdAt": "2025-01-01 12:00"
}
```
상세 응답 예시입니다.

## 3) 이미지 열람
- `originalUrl`, `resizedUrl`을 브라우저에서 열어 최종 확인합니다.
