# 5.10 정리: React 연동으로 프로젝트 마무리

> 목표: React 연동은 참고용으로만 언급하고, 프로젝트를 마무리합니다.

## 목차
1) 프로젝트 마무리
2) React 연동(참고용)
3) 팁: CORS 설정

## 1) 프로젝트 마무리
- Presigned 발급 → S3 업로드 → Lambda 처리 → Spring 메타 저장 → 목록/상세 조회까지 흐름을 완주했습니다.
- 이 장은 **서버 중심 흐름**을 먼저 완성한 뒤, 프론트 연동은 선택지로 남기는 방식으로 마무리합니다.

## 2) React 연동(참고용)
- 리액트 깃: `react-presign-url`
- 이 레포는 **React ↔ Spring 연동 확인**에만 집중합니다.
- 더 공부하고 싶은 사람만 참고해서 사용하세요.

## 3) 팁: CORS 설정
React 개발 서버에서 Spring API를 호출하려면 CORS 허용이 필요합니다.

(확인/수정) 경로: `spring-presign-url/src/main/java/com/metacoding/spring_presign_url/_core/config/CorsConfig.java`
```java
config.addAllowedOrigin("http://localhost:5173"); // React 개발 서버
```
- 포트가 다르면 해당 주소로 변경하세요.
