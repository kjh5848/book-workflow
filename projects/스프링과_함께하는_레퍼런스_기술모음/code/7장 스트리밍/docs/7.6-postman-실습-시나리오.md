## **7.6 Postman 실습 시나리오**

---

### **1. VOD 업로드 요청**

```
POST http://localhost:8080/upload
Body: form-data
Key: file (Type: File)
```

초보자용 체크리스트:

- Method는 `POST`
- Body 탭에서 `form-data` 선택
- Key는 반드시 `file`
- Type은 `File`
- Value에서 업로드할 mp4 파일 선택

업로드가 완료되면 **202 Accepted**와 함께 `HLS 변환 시작` 메시지가 반환됩니다.

---

### **2. 작업 상태 조회(처리중/완료/실패)**

현재 구현은 별도 상태 API가 없으므로, 아래 방식으로 간접 확인합니다.

```
GET http://localhost:8080/hls/720p/video.m3u8
```

- 200 OK: 변환 완료  
- 404/500: 변환 미완료 또는 오류

비동기 변환이 진행 중일 수 있으므로, 1~2회 정도 다시 요청해 보는 것이 좋습니다.

---

### **3. 재생 URL 확인**

```
http://localhost:8080/hls/720p/video.m3u8
http://localhost:8080/hls/1080p/video.m3u8
```

브라우저에서 열면 m3u8이 내려오며, 플레이어(Hls.js)가 ts를 순차 요청합니다.
