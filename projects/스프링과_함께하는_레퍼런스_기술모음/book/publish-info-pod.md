## 2. 추가정보

### 책소개

Spring Boot를 쓰는 주니어 개발자가 실무에서 마주치는 주변 기술들을 하나씩 핵심 흐름을 직접 만들어보며 두려움을 없애는 책입니다.

코드를 맹목적으로 따라치는 튜토리얼이 아니라 "왜 이 기술이 필요한가"를 이야기로 먼저 보여주고, 기술 파트에서 직접 구현합니다. Docker 컨테이너 운영, OAuth 소셜 로그인, Redis 세션 공유, 파일 업로드, 클라우드 스토리지, 실시간 통신, 영상 스트리밍, 검색 엔진, 메시지 큐까지 — 들어는 봤지만 직접 해본 적 없는 9가지 기술을 스타트업에서 3개월간 성장하는 주니어 개발자의 이야기를 따라가며 익힙니다.

각 챕터는 독립적으로 읽을 수 있으며, 처음부터 순서대로 읽으면 하나의 성장 서사가 됩니다.
