# STEP 6. 프롤로그 — "숲을 보여준다"

> **목적**: 모든 챕터가 끝난 후, 코드 없이 전체를 한 눈에.
> **에이전트**: writer → editor
> **검토**: [감수 검토]
> **산출물**: `book/front/prologue.md`

## 흐름

```
1. writer 에이전트 → 각 챕터에서 핵심 개념/비유만 추출
2. 시나리오 흐름대로 하나의 이야기로 연결 → 프롤로그
    ↓
[감수 검토]
  - 기술 감수자: 핵심 개념이 빠지진 않았나?
  - 독자 대변인: 프롤로그만 읽고 책을 사고 싶어지나?
  - 이야기 편집장: 하나의 완결된 이야기인가?
    ↓
저자 확인 → book/front/prologue.md 생성
```

## 프롤로그 작성 규칙

### 기본 규칙
- **코드 없음**: 프롤로그에는 코드가 등장하지 않는다
- **비유만 사용**: 각 챕터의 핵심 비유를 하나의 이야기로 엮는다
- **전체 여정 미리보기**: 독자가 앞으로 어떤 여정을 떠나는지 보여준다
- **호기심 유발**: 읽고 나면 본문이 궁금해져야 한다
- **분량**: A4 2페이지 이상 (약 100-120줄). 각 챕터의 비유를 풍부하게 풀어야 하므로 짧게 끝내지 않는다

### 프롤로그의 목적 (우선순위 순)

프롤로그는 "이 책을 계속 읽게 만드는 것"이 유일한 목적이다.

1. **공감**: 독자가 "나도 저랬는데" 하는 순간이 있어야 한다. 주인공이 막히고 헤매는 감정을 충분히 보여줘야 독자가 끌려 들어온다
2. **궁금증**: 프롤로그만 읽고 본문을 펼치고 싶어야 한다. 역순 훅이 효과적 — "저 사람은 어떻게 저기까지 갔지?"
3. **신뢰**: 비유가 정확하면 "이 사람 진짜 해본 사람이구나" 신뢰가 생긴다
4. **약속**: "읽으면 이걸 얻는다"가 명확해야 한다. 모호한 약속("다양한 기술을 배울 수 있습니다")은 힘이 없다

### 서사 기법
- **역순 훅**: 여정의 끝(성장한 모습)을 먼저 보여주고 첫날로 돌아간다. 같은 자리, 다른 감정으로 수미상관
- **문제 체인**: 기술을 나열하지 않는다. "이걸 풀었더니 저 문제가 터졌다" 식으로 연쇄시킨다 (예: Docker → 서버 2대 → 세션 문제 → Redis)
- **비유는 대화에서 발견**: 설명 → 비유 순서가 아니라, 선배/팀장의 한마디가 비유를 열어주는 구조
- **변화 약속**: 독자에게 "이 책을 읽으면 어떻게 달라지는가"를 명시한다. "문제를 보면 기술이 떠오르는 감각"
- **나열 금지**: 9개 기술을 주차별로 순서대로 나열하면 리스트가 된다. 테마별로 묶거나 문제 체인으로 연결한다
- **감정 밀도**: 분량을 늘릴 때 기술 추가가 아니라 주인공의 감정/독백을 더 넣는다. 막히고 헤매는 과정에 독자가 공감할 틈을 준다

### 소설 작법 체크리스트

프롤로그는 소설의 첫 장이다. 아래 기법을 반드시 적용한다.

| 기법 | 규칙 | 검증 기준 |
|------|------|----------|
| **Show, Don't Tell** | 감정을 직접 서술하지 않고 행동/장면으로 보여준다. "뿌듯했습니다" 대신 "동료 자리로 걸어가서 화면을 보여줬습니다" | 감정 형용사 단독 문장 0건 |
| **감각 묘사** | 시각 + 청각/촉각 중 1개 이상 조합. 모니터 화면, 키보드 소리, 손가락의 감각 등 | 섹션(---구간)당 감각 묘사 1개 이상 |
| **Try/Fail** | 성공 전 최소 2회 실패. 환각→프롬프트 스터핑→실패→RAG처럼 연쇄 | 첫 번째 시도에서 성공하는 장면 0건 |
| **문장 리듬** | 초단문(3~7글자)과 장문(35글자+)을 교차. 같은 어미 3연속 금지 | 초단문 5개 이상, 3박자 0건 |
| **장면 전환** | `---` 앞 마지막 문장의 이미지를 다음 섹션 첫 문장에 겹치거나 연결 | `---` 전후가 완전히 단절된 곳 0건 |
| **구체적 디테일** | 시간("새벽 2시"), 장소("퇴근길 지하철"), 물리적 행동("화이트보드에 화살표를 그렸다") | 섹션당 구체적 디테일 1개 이상 |
| **AI 선호어 금지** | "비로소", "드디어", "마침내", "진정한" 사용 금지 | 0건 |
