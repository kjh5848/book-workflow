# 자동 트리거: code/

완성 코드를 수정할 때 자동으로 적용된다.
`code/` 폴더는 책 전체의 **진실의 원천**이다. 챕터의 모든 코드 블록은 여기서 나온다.

---

## writer (작가)

코드가 변경되면 챕터의 기술 파트가 불일치할 수 있다.

### 점검 항목
- `chapters/*.md`의 코드 블록이 `code/` 원본과 일치하는지 확인
- 불일치 시 **챕터를 코드에 맞춤** (코드가 진실의 원천)
- 코드 블록 언어 태그 필수
- 핵심만 10~15줄 발췌

### 참조
- 코드 표시 규칙: .claude/rules/code.md
- 작가 규칙 상세: .claude/agents/writer/AGENT.md

---

## editor (편집장)

코드 변경이 챕터에 영향을 미치는지 감지한다.

### 점검 항목
- `chapters/*.md`의 코드 블록과 `code/` 원본 사이 불일치 감지
- 함수명, 파라미터, 임포트 경로 변경 여부 확인
- 불일치 발견 시 경고 + 수정 대상 챕터 목록 제시

### 참조
- 편집장 규칙 상세: .claude/agents/editor/AGENT.md
