# 출판정보 생성 스킬

## 로드 시점

PDF 빌드 완료 후, `출판정보 생성` 명령으로 실행.
publisher 에이전트가 PDF 빌드 파이프라인 완료 후 자동 호출.

## 호출 방식

```
출판정보 생성          → POD + 전자책 두 벌 생성
출판정보 생성 POD      → B5(188x257mm)만
출판정보 생성 전자책    → A4(210x297mm)만
```

## 판형 상수

| 유형 | 판형명 | 크기(mm) |
|------|--------|----------|
| POD | 4x6배판(B5) | 188 x 257 |
| 전자책 | 국배판(A4) | 210 x 297 |

## 자동 추출 소스

| 항목 | 소스 | 추출 방법 |
|------|------|----------|
| 책제목 | `planning/seed-v*.md` | `## 주제` 섹션 첫 줄 |
| 부제목 | `planning/seed-v*.md` | 주제 줄에서 `—` 뒤 부분 |
| 목차 | `chapters/*.md` | h1~h3 헤딩 추출 |
| 페이지수 | `book/output/*.pdf` | PDF 페이지 카운트 (`python3 -c "..."`) |
| 책소개 | `planning/seed-v*.md` | `## 책의 의도` + `## 차별점` 합성 |
| 키워드 | `planning/seed-v*.md` | `## 핵심 메시지` + 각 챕터 핵심 개념 |
| 본문언어 | 고정 | 한국어 |

## 빈칸 항목 (유저 직접 입력)

- 저자명, 판유형(초판), 가격, 발행일, 제본형태
- 저자소개, 서평, 표지 이미지, 판권지

## 실행 흐름

```
1. seed-v*.md 최신 버전 읽기 → 책제목, 부제목, 책소개, 키워드 추출
2. chapters/*.md h1~h3 추출 → 목차 생성
3. book/output/*.pdf 페이지 카운트 → 페이지수
   (PDF 없으면 outline 분량 합산으로 대체)
4. 판형 상수 적용 (POD/전자책)
5. 템플릿에 채워서 markdown 생성
```

## 산출물

```
projects/[책이름]/book/publish-info-pod.md      ← POD(B5)
projects/[책이름]/book/publish-info-ebook.md    ← 전자책(A4)
```

## 산출물 템플릿

```markdown
# 출판예정도서 정보

| 항목 | 내용 |
|------|------|

| **부제목** | {자동: seed 부제} |
| **저자역할/저자명** | 저자 / |
| **판유형** | 초판 |
| **크기 (판형)** | {자동: 판형명} ({가로}mm x {세로}mm) |
| **페이지수** | {자동: PDF 페이지 카운트} |
| **제본형태** | |
| **가격** | |
| **발행일** | |
| **본문언어** | 한국어 |
| **키워드** | {자동: seed 키워드; 세미콜론 구분} |

---

## 2. 추가정보

### 책소개

{자동: seed 의도 + 차별점}

### 목차

{자동: chapters h1~h3}

### 저자소개

(직접 입력)

### 서평

(직접 입력)

### 표지/표제지

(이미지 파일 첨부)

(직접 입력)
```

## 목차 추출 규칙

- `chapters/*.md` 파일을 번호순으로 읽는다
- `#` (h1), `##` (h2), `###` (h3) 헤딩만 추출
- `## 이야기 파트`, `## 기술 파트` 등 파트 구분 h2는 제외
- `<!-- -->` 주석 내부의 헤딩은 제외
- 들여쓰기로 계층 표현

```
Ch.1: Hallucination과 RAG
  이야기 파트
    1.1 입사 3일 차, 첫 번째 임무
    1.2 LLM의 자신감 넘치는 거짓말
    ...
```

## 페이지 카운트

```bash
python3 -c "
import subprocess
result = subprocess.run(['typst', 'query', 'book.typ', '<page>', '--field', 'number'], capture_output=True, text=True)
# 또는 PyPDF2/pikepdf로 PDF 페이지수 확인
"
```

PDF가 없으면 `outline-v*.md`의 `예상 분량` 합산값을 `(추정)` 표시와 함께 사용.
