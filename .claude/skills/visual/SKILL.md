# 시각 요소 스킬

## 로드 시점

| STEP | 상황 |
|------|------|
| STEP 3. 시나리오+버전 | 버전별 구조 다이어그램 |
| STEP 4. 뼈대 | 파일 계층 트리, 난이도 곡선 |
| STEP 5. 챕터 집필 | 개념 다이어그램, 누적 구조도, 이미지 플레이스홀더 삽입 |

## 핵심 규칙

### Mermaid
- **`flowchart`** 키워드 (`graph` 아님)
- 노드: **최대 7개** — 넘으면 subgraph로 분리
- 특수문자 라벨: **큰따옴표 필수**
- 엣지 라벨: `A -- "텍스트" --> B` 형식
- **Bold/Italic (`**`, `*`): Mermaid 안에서 절대 금지**

### 이미지
- 캡션: `*그림 {장}-{순번}: 설명*` 형식
- 집필 시 **플레이스홀더 삽입** → 실제 이미지는 집필 완료 후 생성
- 플레이스홀더 2종: `[GEMINI PROMPT: ...]` (개념도) / `[CAPTURE NEEDED: ...]` (실습 캡처)
- 이미지 경로: `assets/CH{N}/` 챕터별 폴더
- **스크린샷과 코드 블록 중복 금지**: 스크린샷이 있으면 출력 코드 블록 제거

## 담당 스킬

### Tier 2 (입출력 정의)

| 스킬 | 입력 → 출력 |
|------|------------|
| B4. 계층-생성기 | 태그 붙은 파일 목록 → 트리 시각화 (태그 + 한 줄 설명) |
| B5. 난이도-곡선기 | 챕터 목록 + 개념 수 → 난이도 곡선 + 스파이크 경고 |
